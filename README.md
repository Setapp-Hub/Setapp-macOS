## About Setapp

**Setapp** — the ultimate Mac app subscription: Free, 240+ handpicked premium apps, zero ads, always up-to-date.  
From productivity monsters (CleanMyMac X, Ulysses, Bartender) to creative tools (Pixelmator Pro, Aeon Timeline) and dev essentials (TablePlus, Paw, iStat Menus).

This community toolkit adds **custom configs**, **exclusive templates**, **keyboard shortcuts**, and **workflow automations** that official apps don’t include.
